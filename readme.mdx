# API - Quarto Digital

## O que é?

É uma Api desenvolvida como TCC do curso de Análise e Desenvolvimento de Sistemas da UPF. 
Que tem como objetivo a criação de um sistema de gerenciamento de quartos de hotel. 

## Tecnologias utilizadas

- [Node.js](https://nodejs.org/en/) - Interpretador de JavaScript assíncrono com código aberto.
- [Express](https://expressjs.com/pt-br/) - Framework para Node.js que fornece recursos mínimos para construção de aplicações web.
- [PostgreSQL](https://www.postgresql.org/) - Sistema de gerenciamento de banco de dados objeto relacional.
- [Prisma](https://www.prisma.io/) - ORM para Node.js e TypeScript.
- [TypeScript](https://www.typescriptlang.org/) - Superset de JavaScript que adiciona tipagem estática e alguns outros recursos a linguagem.
- [Jest](https://jestjs.io/) - Framework de testes para JavaScript.
- [JWT](https://jwt.io/) - JSON Web Token é um padrão aberto (RFC 7519) que define uma forma compacta e independente de transmitir informações de forma segura entre partes como um JSON Object.
- [Bcrypt](https://www.npmjs.com/package/bcrypt) - Biblioteca para criptografia de senhas.
- [NodeMailer](https://nodemailer.com/about/) - Biblioteca para envio de e-mails.

## Instalação

Para instalar o projeto, é necessário ter o [Node.js](https://nodejs.org/en/) instalado em sua máquina.

Após clonar o repositório, execute o comando abaixo para instalar as dependências do projeto.

```bash
npm install
```

## Configuração

Para configurar o projeto, é necessário criar um arquivo .env na raiz do projeto e preencher as variáveis de ambiente.

```bash
# .env
PORT=3000
DATABASE_URL=postgresql://user:password@localhost:5432/database?schema=public
JWT_SECRET=secret
EMAIL=smtp.gmail.com
EMAIL_PASSWORD=password
```

## Execução

Para executar o projeto, execute o comando abaixo.

```bash
npm run dev
```

## Testes

Para executar os testes, execute o comando abaixo.

```bash
npm run test
```
